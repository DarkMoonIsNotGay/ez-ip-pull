How to easily pull ips with this simple script
Just paste your webhook and enjoy ips.

For better hide you can use PyArmor
- pyarmor gen (name of the script)[ip.py]

Or

Use py2exe 
- build_exe (name of the script)[ip.py]

You will need:
  - requests
  - pyarmor (optional)
  - py2exe (optional)
